Help: wgTimelines

Description

The wgTimelines module can be used to modules in XOOPS

Install/uninstall

No special measures necessary, follow the standard installation process and extract the wgtimelines folder into the ../modules directory. Install the module through Admin -> System Module -> Modules.

Detailed instructions on installing modules are available in the XOOPS Operations Manual

Features

Basics:

  • You can define one or more timelines
  • You can define one or more items per timeline
  • You can adopt the templates corresponding your wishes
Styles:
  • Varios table styles
  • Various image styles
  • Varios posibilities for positioning a images
  • and so one
